About

ProjectOscar (Open Soure Computed Audio Robot) is a project that was dreamt up by developer tscrip. He invisioned a world where technologies such as Siri ® or Google Now ® were open source platforms that allow anyone extend functionality. Currently Oscar is composed of two pieces.

  • Oscar Core is a Python based webserver that responds to requests from requests.
  • The multiplatform application that is used to initiate requests to Oscar.
